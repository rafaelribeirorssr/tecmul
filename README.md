## Sobre o Jogo

**Cyber Escape** é um endless runner 3D desenvolvido em Unity, com uma estética neon. Controlas um hacker que foge a alta velocidade por uma cidade cibernética, desviando-se de firewalls, saltando por cima de antivirus e recolhendo dinheiro roubado enquanto a velocidade aumenta progressivamente.

Quanto mais longe chegares, mais difícil fica. Consegues escapar?

---

##  Como Jogar

| Ação | Tecla |
|------|-------|
| Mover para a direita | `D` ou `→` |
| Mover para a esquerda | `A` ou `←` |
| Saltar | `Espaço`, `W` ou `↑` |

- **Evita** os obstáculos e barreiras na tua rota
- **Recolhe** o dinheiro espalhado pelo caminho
- **Sobrevive** o máximo de tempo possível a velocidade aumenta continuamente!

---

##  Funcionalidades

-  Sistema de **3 pistas** (lanes) para esquivar
-  Velocidade **progressiva** fica cada vez mais difícil
-  Sistema de **pontuação** por dinheiro recolhido
-  Ambiente **cyberpunk/neon** imersivo
-  Música e efeitos sonoros
-  **Game Over** ao colidir com obstáculos
-  Sistema de **retry** e menu principal

---

##  Tecnologias Utilizadas

- **Motor:** Unity 6.3 LTS
- **Linguagem:** C#
- **Input System:** Unity New Input System
- **Física:** Unity Rigidbody
- **Animações:** Unity Animator Controller
- **Modelos 3D:** FBX / GLB
- **Controlo de versão:** Git + GitHub
